# Icon Placeholder

The manifest.json references icon files in this directory:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels) 
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

For testing, you can create simple PNG icons or the extension will work without icons (Chrome will use default).

To test the extension:
1. Open Chrome and go to chrome://extensions/
2. Enable "Developer mode" in the top right
3. Click "Load unpacked" and select the /extension folder
4. The extension should now be loaded and ready for testing
